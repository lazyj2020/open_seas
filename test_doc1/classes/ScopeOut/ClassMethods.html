method_missing_with_scope_out(method_called, *args, &block)

allow scope_out users to be able to do cool stuff like Student.find_all_active_by_age(18) this will use the with_active scope and then pass :find_all_by_age up the method_missing chain

scope_out(name, options = {}, &block)

given a name such as ‘active’, scope_out creates 3 methods 

  with_active creates a scope
  find_active uses the with_active scope to call find
  calculate_active uses the with_active scope to call calculate
